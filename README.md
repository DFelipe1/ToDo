<h2 id="started">🚀 Introdução</h2>

Aumente sua produtividade gerenciando suas tarefas e aumentando sua performance.

<h3>Pré-requisitos</h3>

Aqui você lista todos os pré-requisitos necessários para execução do seu projeto. Por exemplo:

- [NodeJS](https://nodejs.org/pt)
- [Git 2](https://git-scm.com)

<h3>Clonando</h3>

Como fazer para clonar o projeto

```bash
git clone https://github.com/DFelipe1/ToDo
```

<h3>Inicie o projeto</h3>

start fornt-end:
```bash
cd /web
npm install
npm run dev 
```

start back-end:
```bash
cd /server
npm install
npm run dev 
```

veja a documentação do back e front end para entender melhor:

- [Back-end](./server/README.md)
- [Front-end](./web/README.md)
